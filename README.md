# Analyse des ventes d’un e-commerce

## Contexte et objectifs

![Logo GRANDMARCHE](P2_logo_GRANDMARCHE.jpg)  
L’objectif est d’analyser les performances d’un site e-commerce afin d’identifier les tendances clés, les meilleures ventes, la saisonnalité et les axes d’optimisation commerciale.  
Le projet se concentre sur la visualisation des données et la narration analytique.

## Soft Skills
- Présenter des résultats de manière claire et structurée  
- Choisir les graphiques les plus adaptés au message  
- Raconter une histoire à partir des données (data storytelling)  
- Synthétiser des insights actionnables  

## Compétences
- Nettoyer et structurer des données sous Excel  
- Utiliser les tableaux croisés dynamiques  
- Construire des visualisations pertinentes  
- Analyser les ventes : saisonnalité, top produits, panier moyen  

## Outils utilisés
- Excel  
- PowerPoint (support de présentation)
